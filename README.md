# DISMD RSVP Engine (V3)

A lean, serverless web utility built to eliminate boring, static invitation vibes through immediate interactive experiential design. 

## 🚀 Architecture Overview
- **Frontend:** Single-file standalone HTML5/CSS3 application layout.
- **Backend Infrastructure:** Serverless multi-tenant database integration via Supabase REST API endpoints.
- **Hosting Tier:** 100% unmetered free-forever static hosting via GitHub Pages.

## 🛡️ AI Governance & Security
- **Row-Level Security (RLS):** Enabled to ensure safe, public write/read transactions.
- **Credential Isolation:** Public token variables are bound exclusively to client-side network headers; all administrative secret keys are explicitly restricted from the source control environment.

## 🔮 Future Roadmap
- Platform migration to Lovable for advanced UI/UX component compilation.
- Real-time conversational voice synthesis integration via WebRTC streaming pipelines.
